**PHPをMongodbに接続する**


➀ドライバのインストール


```
pecl install mongodb
```


最後に、以下のコードをphp.iniファイルに加える。


```
extension=mongodb.so
```


➁MongoDB Altasに接続する


```PHP
<?php
$client = new MongoDB\Client(
    'mongodb+srv://<username>:<password>@<cluster-address>/test?retryWrites=true&w=majority'
);

```


```
$db = $client->test;
```
