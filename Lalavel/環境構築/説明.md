基本的に自分が知る限りPHPをローカル環境で実行するにはnginx(エンジンエックス)かApaccheを使うかの二択だ。
今回はローカルにPHP環境があることを前提として話していく。
とりあえずコマンドプロンプトで以下のコードを実行する。
```
php -version
```
これを実行したときに、バージョン情報が表示されれば大丈夫だ。
つぎに、phpのバージョン管理アプリをインストールする。
[インストールページ](https://getcomposer.org/download/)
その後、以下のコードを実行する
```
composer create-project "laravel/laravel=5.7.*" プロジェクト名
```
こうすればよい。
そしてプロジェクトディレクトリに移動し、
```
php artisan serve
```
を実行してサーバを立てる。
